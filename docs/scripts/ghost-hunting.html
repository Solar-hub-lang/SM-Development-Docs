Ghost Hunting

SM Development Ghost Hunting er et multiplayer job, hvor spillere kan jage ghosts, bruge target/progress, køre ud til locations og optjene rewards/leaderboard progress.

JobMultiplayerox_targetLeaderboard

Features

  • Multiplayer job for fx 2-6 spillere.
  • Timer på missionen.
  • Ghost locations i config.
  • Passive og aggressive ghosts.
  • Fang spøgelse interaction via target.
  • Progress UI.
  • Leaderboard/stat tracking.
  • Vehicle spawn til holdet.
  • Reward efter mission.

Dependencies

Installation

Upload resource

Upload resource til resources/[SM]/SM_GhostHunting.

Importér SQL

Hvis leaderboard bruges, importér SQL filen.

Ret config

Indstil start location, ghost coords, vehicle spawn, timer, team size og rewards.

Start resource

Tilføj resource i server.cfg efter dependencies.

ensure ox_lib
ensure ox_target
ensure es_extended
ensure SM_GhostHunting

Config eksempel

Config.Command = 'ghosthunt'
Config.MinPlayers = 2
Config.MaxPlayers = 6
Config.MissionTime = 15 * 60

Config.StartLocation = vec3(-258.4, -975.6, 31.2)
Config.Vehicle = 'rumpo'
Config.VehicleSpawn = vec4(-263.2, -982.8, 31.2, 250.0)

Config.Ghosts = {
    {
        label = 'Abandoned House',
        coords = vec3(1002.5, -2145.2, 30.5),
        type = 'aggressive'
    },
    {
        label = 'Old Motel',
        coords = vec3(326.7, -213.1, 54.1),
        type = 'passive'
    }
}

Gameplay flow

  1. Spillere starter jobbet ved start NPC/location.
  2. Holdet får en mission og eventuelt et køretøj.
  3. Ghosts spawner på valgte locations.
  4. Spillere bruger target/interact for at fange ghosts.
  5. Missionen slutter når timeren udløber eller alle ghosts er fanget.
  6. Rewards og leaderboard opdateres serverside.

Fejlfinding

  • Hvis ghosts ikke spawner, tjek model load og coords.
  • Hvis target ikke virker, tjek ox_target og entity registration.
  • Hvis vehicle ikke spawner, tjek modelnavn og spawn coords.
  • Hvis keys ikke gives, tjek om t1ger_keys eventnavn matcher.
  • Hvis leaderboard fejler, tjek SQL og oxmysql.
