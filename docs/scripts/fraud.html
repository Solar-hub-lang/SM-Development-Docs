Fraud

SM Development Fraud er et FiveM fraud/criminal activity script med startpunkt, tilfældige locations, required items, rewards, cooldowns, police alerts og mulighed for ATM/skimmer gameplay.

Criminal JobESXox_inventoryPolice Alerts

Features

  • Start fraud fra en config location.
  • Tilfældige fraud locations fra Config.FraudLocations.
  • Required items før start.
  • Cooldown og timeout.
  • Reward valideret serverside.
  • Police alerts.
  • Area/location updates.
  • ATM/skimmer extension mulighed.
  • Locales, fx dansk og engelsk.

Installation

Upload resource

Upload resource til resources/[SM]/SM_Fraud.

Tilføj items

Hvis scriptet kræver items, skal de oprettes i dit inventory.

Ret config

Indstil start location, fraud locations, rewards, cooldowns, required items og police alerts.

Start scriptet

Tilføj ensure SM_Fraud i server.cfg.

ensure ox_lib
ensure ox_inventory
ensure es_extended
ensure SM_Fraud

Config eksempel

Config.FraudLocation = vec3(127.1, -1710.2, 29.3)

Config.RequiredItems = {
    { item = 'laptop', amount = 1 },
    { item = 'usb', amount = 1 }
}

Config.FraudLocations = {
    vec3(245.2, -1010.4, 29.3),
    vec3(-1204.4, -326.3, 37.8),
    vec3(313.9, -280.5, 54.1)
}

Config.Cooldown = 20 * 60
Config.Timeout = 15 * 60

Config.Rewards = {
    { item = 'black_money', min = 2500, max = 6500 }
}

Security krav

Server-side validation:

Clienten må ikke sende item, amount, price eller reward direkte. Serveren skal selv finde reward fra config og tjekke distance, cooldown og items.

ATM/skimmer extension

  • Skimmer item kan installeres på ATM.
  • Spiller venter, indsamler data og leverer til NPC.
  • Memory game eller hack minigame kan bruges.
  • Rewards skal gives serverside med CanCarryItem check.

Fejlfinding

  • Hvis rewards ikke gives, tjek item-navne og ox_inventory.
  • Hvis politiet ikke får alert, tjek webhook/job name/event.
  • Hvis spiller kan exploite rewards, tjek at reward beregnes serverside.
  • Hvis cooldown ikke virker, tjek activeFrauds/cooldown table serverside.
