Admin Panel

SM Development Admin Panel er et premium ingame adminpanel til FiveM servere med staff login, permissions, reports, dev tools, Discord integration og license system.

AdminReportsPermissionsLicenseDiscord Bot

Features

  • Staff login med roller og permissions.
  • Owner/master account.
  • Player list og spillerprofil.
  • Bring, goto, freeze, kill, revive og heal.
  • Kick, ban, tempban, warn og unban.
  • Reports UI for spillere via /report.
  • Adminpanel med transparent slider.
  • Developer tab med resource status og fejlvisning.
  • Audit logs.
  • License/IP-lock system.
  • Discord bot integration til license og staff verification.

Dependencies

DependencyPåkrævetBruges til

Installation

Upload resource

Upload adminpanelet til resources/[SM]/SM_Admin eller behold resource-navnet, hvis pakken bruger NexusRP_Admin.

Importér SQL

Importér alle SQL tables til din FiveM database.

Ret config

Indsæt license key, admin command, report command, Discord settings og permissions.

Start resource

Tilføj resource til server.cfg efter dependencies.

ensure oxmysql
ensure ox_lib
ensure es_extended
ensure SM_Admin

Config eksempel

Config.Command = 'admin'
Config.ReportCommand = 'report'
Config.Branding = 'SM Development'
Config.Debug = false

Config.License = {
    Enabled = true,
    Key = 'DIN_LICENSE_KEY_HER'
}

smlicensecheckConsoleTvinger license check.

Report UI krav

  • /report skal kun åbne player report UI.
  • Report UI skal ikke åbne adminpanelet.
  • Report UI skal kunne lukkes med ESC, X, Annuller og klik på baggrund.
  • Report UI skal ikke have blur i baggrunden.

Developer tab

Developer tabben bruges til resource status, script-fejl, error logs og hurtig debugging.

Fejlfinding

  • Hvis adminpanelet åbner sammen med /report, er report UI og admin UI ikke adskilt korrekt i NUI.
  • Hvis staff ikke kan logge ind, tjek database, password, Discord role verification og permissions.
  • Hvis license fejler, tjek license key, IP-lock og license API.
