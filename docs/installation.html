Installation

Denne guide viser den normale måde at installere SM Development scripts på en FiveM server.

FiveMESXox_liboxmysql
Vigtigt:

Upload aldrig license keys, database passwords, bot tokens eller webhooks offentligt på GitHub.

Standard installation

Download scriptet

Download zipfilen fra SM Development og pak den ud på din computer.

Upload resource

Upload script-mappen til din servers resources mappe, fx resources/[SM]/SM_Admin.

Installer dependencies

Sørg for at nødvendige dependencies som es_extended, ox_lib, oxmysql og eventuelt ox_inventory er installeret.

Importér SQL

Hvis scriptet har en .sql fil, skal den importeres i din database via phpMyAdmin eller databaseværktøj.

Ret config.lua

Åbn config.lua og indsæt license key, webhooks, jobs, items, commands og andre settings.

Tilføj i server.cfg

Tilføj scriptet efter dependencies i server.cfg.

server.cfg eksempel

ensure oxmysql
ensure ox_lib
ensure es_extended

ensure SM_Admin
ensure SM_Fraud
ensure SM_GhostHunting

Resource rækkefølge

Dependencies skal altid starte før selve scriptet. Hvis et script bruger ox_inventory, skal inventory også startes før scriptet.

TypeStartes førEksempel
DatabaseAlle scripts med SQLensure oxmysql
FrameworkESX scriptsensure es_extended
LibraryScripts med UI/notify/callbackensure ox_lib
InventoryScripts med itemsensure ox_inventory

Efter installation

  • Genstart serveren.
  • Tjek server console for errors.
  • Tjek F8 console ingame, hvis UI ikke åbner.
  • Test alle commands som staff/admin.
  • Test scriptet med en normal spiller uden admin permissions.
