License System

Flere SM Development scripts kan bruge license-system med license key, API check og IP-lock.

SecurityLicense KeyIP-lockDiscord Bot

Hvad license systemet tjekker

  • Om license keyen findes.
  • Om license keyen er aktiv.
  • Om licensen er udløbet.
  • Om licensen er disabled eller banned.
  • Om serverens IP matcher licensen.
  • Om license API’en svarer korrekt.

Typisk config

Config.License = {
    Enabled = true,
    Key = 'DIN_LICENSE_KEY_HER',
    Debug = false
}

Config.LicenseAPI = {
    Url = 'https://your-license-api.example.com',
    Timeout = 5000
}

Console commands

Commands kan variere mellem scripts, men kan fx være:

smlicensestatus
smlicensecheck
nexuslicensestatus
nexuslicensecheck

Discord bot commands

/license-create
/license-check
/license-disable
/license-enable
/license-ban
/license-delete
/license-reset-ip
/license-list
Upload aldrig secrets.

Bot token, API secret, database password, webhook URL og license keys må ikke ligge offentligt på GitHub.

Når license fejler

  1. Tjek om keyen er skrevet korrekt.
  2. Tjek om serverens IP matcher licensen.
  3. Tjek om API’en er online.
  4. Tjek om serveren har internetadgang.
  5. Tjek om licensen er expired, disabled eller banned.
