Troubleshooting

Brug denne side når et SM Development script ikke starter, UI ikke åbner, database fejler eller license ikke virker.

Scriptet starter ikke

  • Tjek resource-navnet i server.cfg.
  • Tjek at alle dependencies starter før scriptet.
  • Tjek at SQL er importeret.
  • Tjek om config.lua har syntax errors.
  • Tjek server console for den første error, ikke kun den sidste.

Database fejl

Table 'database.table_name' doesn't exist

Det betyder normalt, at SQL-filen ikke er importeret, eller at du bruger forkert database.

NUI åbner ikke

  • Tjek at fxmanifest.lua har korrekt ui_page.
  • Tjek at html/index.html findes.
  • Tjek F8 console for JavaScript errors.
  • Tjek om NUI callbacks matcher mellem client.lua og app.js.

License invalid

  • Key er forkert.
  • IP-lock matcher ikke.
  • License API er offline.
  • License er expired, disabled eller banned.

Bot starter ikke

npm install
node index.js
  • Tjek Node.js version.
  • Tjek bot token.
  • Tjek database config.
  • Tjek Discord bot permissions.

Support ticket

Når du laver en ticket, send:

  • Scriptnavn.
  • Server console error.
  • F8 console error.
  • Hvad du gjorde før fejlen kom.
  • Din framework/inventory/database setup.
Send aldrig:

License key, bot token, database password, webhook URL eller panel secret.
